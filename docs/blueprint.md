# **App Name**: LexCase Manager

## Core Features:

- Role-Based Access Control: User authentication with admin and lawyer roles, with different permission levels. Admin manages all cases and users. Lawyers manage their assigned cases only.
- Case Overview: Display cases with details like NUREJ, client name, cause, process stage, next activity, and subject. Order cases by most recent activity.
- Reminder Scheduling: Allow users to schedule and manage reminders related to their cases.
- OneDrive Integration: Store links to documents in OneDrive.
- AI-Powered Push Notifications: The AI tool generates push notifications, leveraging reasoning to determine when to alert the lawyer or administrator. Generates notifications based on Firebase.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and professionalism, suitable for legal environments.
- Background color: Light gray (#F5F5F5), providing a neutral backdrop that ensures readability and focuses attention on content.
- Accent color: Soft purple (#9575CD), analogous to blue but contrasting enough to highlight key interactive elements, avoiding stark color clashes.
- Clean and professional typography for readability.
- Use clear and intuitive icons for case statuses and actions.
- Clear layout to highlight relevant content, like dates, NUREJ numbers and assigned lawyers.
